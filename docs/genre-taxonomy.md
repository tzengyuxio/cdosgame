# Genre Taxonomy (v2)

cdosgame 的遊戲類型分類。**正本定義在此**；`schema/game.schema.mjs` 的 `GENRES` 列舉 key、
`src/lib/labels.js` 持有顯示名與 group 對應。

## 存值原則

- **frontmatter 的 `genre` 欄位存「key」**（如 `SLG`），不是中文名。key 穩定，中文顯示名日後可改而不動 content。
- 每款只給**一個** key（或 `null` 表未分類）。group 不入檔，由 `GENRE_GROUPS[key]` 推導。
- 顯示一律經 `genreLabel(key)`；分組瀏覽經 `genreGroup(key)` → `groupLabel(gid)`。

## 判準（避免重判時搖擺）

- **SLG vs HSG**：玩法是「武將登用＋內政外交＋會戰」的君主經營（光榮 like）→ **HSG**，題材不限；否則回合策略 → **SLG**。「歷史相關但非君主經營」（如文明帝國的泛文明科技樹）一律 SLG。
- **SLG/HSG vs RTS**：即時操作一律 → **RTS**，不論題材。
- **ACT vs AADV**：純關卡動作 → ACT；探索導向 → AADV。
- **SIM vs LSG vs CMS**：擬真載具 → SIM；養人/人生 → LSG；商業經營 → CMS。
- **戀愛遊戲分流**：有「參數養成＋時間/行動管理」骨架 → **LSG**（心跳回憶/純愛手札式）；純讀劇情選分支、無數值經營 → **AVG**。
- **group 歸屬按「名稱主詞」**：SRPG/ARPG 主詞 RPG → 角色扮演；AADV 主詞冒險 → 冒險。
